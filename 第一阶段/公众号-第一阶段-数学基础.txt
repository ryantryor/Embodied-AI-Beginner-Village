一、线性代数：控制工程的语言

▶ 1.1 为什么线性代数这么重要？

在现代控制理论中，我们用状态空间来描述系统：

状态方程：ẋ = Ax + Bu
输出方程：y = Cx + Du

这里的 A、B、C、D 全都是矩阵！

如果你想：
• 判断系统稳定性 → 需要计算矩阵的特征值
• 分析系统能控性 → 需要计算矩阵的秩
• 设计最优控制器 → 需要解Riccati方程（矩阵方程）
• 实现卡尔曼滤波 → 需要做矩阵分解

可以说，不懂线性代数，就学不好现代控制理论。

━━━━━━━━━━━━━━━━━━━━━━

▶ 1.2 矩阵基础

【矩阵是什么？】

矩阵就是一个数字表格：

A = [a11  a12]
    [a21  a22]

这是一个 2×2 矩阵，有 2 行 2 列。

【矩阵的基本运算】

加法：对应元素相加

[1 2] + [5 6] = [6  8]
[3 4]   [7 8]   [10 12]

乘法：行×列

[1 2] × [5 6] = [19 22]
[3 4]   [7 8]   [43 50]

⚠️ 注意：矩阵乘法不满足交换律！A×B ≠ B×A

【特殊矩阵】

• 单位矩阵 I：对角线为1，其余为0
• 零矩阵 O：所有元素为0
• 对角矩阵：非对角线元素为0
• 转置矩阵 Aᵀ：行列互换
• 逆矩阵 A⁻¹：A×A⁻¹ = I

━━━━━━━━━━━━━━━━━━━━━━

▶ 1.3 行列式

【什么是行列式？】

行列式是一个把矩阵变成一个数的运算。

对于 2×2 矩阵：
det([a b]) = ad - bc
    [c d]

【行列式的意义】

• det > 0：保持方向，放大/缩小
• det < 0：反转方向
• det = 0：压缩到低维（不可逆！）

在控制工程中，det = 0 意味着矩阵不可逆，这是非常重要的判断条件！

━━━━━━━━━━━━━━━━━━━━━━

▶ 1.4 特征值与特征向量（核心中的核心！）

【定义】

对于矩阵 A，如果存在非零向量 v 和标量 λ，使得：

Av = λv

那么：
• λ 称为 A 的特征值
• v 称为 A 的特征向量

【几何意义】

特征向量是矩阵变换下只伸缩、不旋转的方向。
特征值是伸缩的倍数。

【如何计算特征值？】

解特征方程：det(A - λI) = 0

【为什么特征值在控制中如此重要？】

系统的稳定性由矩阵 A 的特征值决定！

ẋ = Ax

• 如果 A 的所有特征值的实部都 < 0，则系统稳定
• 如果有任何一个特征值的实部 ≥ 0，则系统不稳定

这是控制工程中最基本、最重要的结论之一！

━━━━━━━━━━━━━━━━━━━━━━

▶ 1.5 Python 实践：用 NumPy 做矩阵运算

import numpy as np

# 创建矩阵
A = np.array([[1, 2], [3, 4]])

# 矩阵运算
print("A × B =", A @ B)

# 行列式
print("det(A) =", np.linalg.det(A))

# 特征值
eigenvalues, eigenvectors = np.linalg.eig(A)
print("特征值:", eigenvalues)

二、微积分与微分方程

▶ 2.1 导数

【什么是导数？】

导数描述的是函数在某点的瞬时变化率。

f'(x) = lim[f(x+h) - f(x)] / h

几何意义：曲线在某点的切线斜率。

【常用导数公式】

• c（常数）→ 0
• xⁿ → nxⁿ⁻¹
• eˣ → eˣ
• sin(x) → cos(x)
• cos(x) → -sin(x)

━━━━━━━━━━━━━━━━━━━━━━

▶ 2.2 偏导数

【什么是偏导数？】

对于多元函数 f(x, y)，偏导数是固定其他变量，对某一个变量求导。

∂f/∂x = 固定 y，对 x 求导

在控制中，偏导数用于线性化非线性系统！

━━━━━━━━━━━━━━━━━━━━━━

▶ 2.3 微分方程

【什么是微分方程？】

含有未知函数及其导数的方程叫做微分方程。

物理世界的规律大多用微分方程描述：
• 牛顿第二定律：m(d²x/dt²) = F
• 弹簧振动：m(d²x/dt²) + c(dx/dt) + kx = F(t)

━━━━━━━━━━━━━━━━━━━━━━

三、拉普拉斯变换：控制工程师的魔法棒

▶ 3.1 什么是拉普拉斯变换？

拉普拉斯变换是把时域函数变成复频域函数的数学工具。

L{f(t)} = F(s) = ∫₀^∞ f(t)e^(-st) dt

【为什么我们需要拉普拉斯变换？】

在时域解微分方程很麻烦，但在频域：
• 微分方程 → 代数方程
• 卷积 → 乘法
• 系统分析变得简单

━━━━━━━━━━━━━━━━━━━━━━

▶ 3.2 常用拉普拉斯变换对

时域 f(t) → 频域 F(s)

• 1（单位阶跃）→ 1/s
• t → 1/s²
• e^(-at) → 1/(s+a)
• sin(ωt) → ω/(s²+ω²)
• cos(ωt) → s/(s²+ω²)

━━━━━━━━━━━━━━━━━━━━━━

▶ 3.3 拉普拉斯变换的性质

【微分性质（最重要！）】

L{df/dt} = sF(s) - f(0)

这个性质让我们可以把微分方程变成代数方程！

【传递函数】

传递函数是线性系统在零初始条件下，输出与输入的拉普拉斯变换之比：

G(s) = Y(s)/U(s)

━━━━━━━━━━━━━━━━━━━━━━

四、复变函数与频率分析

▶ 4.1 复数基础

【复数的表示】

z = a + bj

• a = 实部
• b = 虚部
• j = √(-1)

【复频域 s】

在拉普拉斯变换中，s 是一个复变量：

s = σ + jω

当 s = jω 时，G(jω) 就是传递函数的频率响应。

五、概率统计基础

▶ 5.1 为什么需要概率统计？

在现实中：
• 传感器测量有噪声
• 系统模型有不确定性
• 外部干扰是随机的

卡尔曼滤波就是用概率统计的方法来处理这些问题。

━━━━━━━━━━━━━━━━━━━━━━

▶ 5.2 重要概念

【期望（均值）】

E[X] = 平均值

【方差】

Var(X) = 数据分散程度

【高斯分布】

f(x) = (1/√(2πσ²)) × e^(-(x-μ)²/(2σ²))

高斯分布是最重要的概率分布：
• 传感器噪声通常服从高斯分布
• 卡尔曼滤波假设噪声是高斯的
