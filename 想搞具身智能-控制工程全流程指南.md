## 第一阶段：数学基础——通往控制世界的钥匙

![数学基础](https://images.unsplash.com/photo-1635070041078-e363dbe005cb?w=800)

控制工程的本质是**用数学语言描述物理世界**。如果你数学基础扎实，后面的学习会事半功倍。

### 1.1 线性代数 ⭐⭐⭐⭐⭐

**为什么重要？**

现代控制理论几乎全部建立在矩阵运算之上。状态空间表示、能控性能观性分析、最优控制——全都离不开线性代数。

**核心内容：**

| 知识点 | 控制工程应用 |
|--------|--------------|
| 矩阵运算 | 状态方程 ẋ = Ax + Bu |
| 特征值与特征向量 | 判断系统稳定性 |
| 矩阵分解 | 卡尔曼滤波、最优控制 |
| 向量空间 | 状态空间分析 |

**学习建议：**

- 📺 **强烈推荐**：3Blue1Brown《线性代数的本质》（B站有中字）
- 📚 教材：Gilbert Strang《线性代数导论》
- 💻 实践：用 Python `numpy` 库动手算

**关键公式：**

```
状态空间表示：
ẋ = Ax + Bu   （状态方程）
y = Cx + Du   （输出方程）

系统稳定性判断：
特征值 λ 的实部全部为负 → 系统稳定
```

---

### 1.2 微积分与微分方程 ⭐⭐⭐⭐⭐

**为什么重要？**

物理世界的运动规律本质上都是微分方程。牛顿第二定律 F=ma 就是一个二阶微分方程。

**核心内容：**

| 知识点 | 控制工程应用 |
|--------|--------------|
| 导数与偏导数 | 系统灵敏度分析 |
| 微分方程 | 系统动力学建模 |
| 拉普拉斯变换 | 传递函数分析 |
| 傅里叶变换 | 频域分析 |

**拉普拉斯变换——控制工程师的"魔法棒"：**

拉普拉斯变换可以把复杂的微分方程变成简单的代数方程：

```
时域（难算）          →    频域（好算）
微分方程              →    代数方程
f(t)                  →    F(s)
```

**常用变换对：**

| 时域 f(t) | 频域 F(s) |
|-----------|-----------|
| 1 | 1/s |
| t | 1/s² |
| e^(-at) | 1/(s+a) |
| sin(ωt) | ω/(s²+ω²) |

---

### 1.3 复变函数 ⭐⭐⭐⭐

**为什么重要？**

控制系统的频率特性分析需要复数运算。波特图、奈奎斯特图都建立在复频域之上。

**核心概念：**

```
复频率：s = σ + jω

传递函数的频率响应：
G(jω) = |G(jω)| · e^(j∠G(jω))
        ↑幅值      ↑相位
```

## 第二阶段：自动控制原理——控制工程的核心

![控制系统](https://images.unsplash.com/photo-1518770660439-4636190af475?w=800)

自动控制原理是控制工程的**入门必修课**，也是浙大控制工程面试的**重点考察内容**。

### 2.1 系统建模 ⭐⭐⭐⭐⭐

**什么是系统建模？**

用数学方程描述物理系统的行为。

**经典案例：弹簧-质量-阻尼系统**

![弹簧质量阻尼系统](https://upload.wikimedia.org/wikipedia/commons/thumb/4/45/Mass_spring_damper.svg/800px-Mass_spring_damper.svg.png)

运动方程：
```
mẍ + cẋ + kx = F(t)

其中：
m = 质量
c = 阻尼系数
k = 弹簧刚度
F(t) = 外力
```

传递函数形式：
```
G(s) = X(s)/F(s) = 1/(ms² + cs + k)
```

---

### 2.2 时域分析 ⭐⭐⭐⭐⭐

**核心问题：系统输入一个信号后，输出会怎样变化？**

**一阶系统：**

```
G(s) = K/(τs + 1)

τ = 时间常数（决定响应快慢）
K = 增益
```

**二阶系统（最重要！）：**

```
G(s) = ωn²/(s² + 2ζωns + ωn²)

ωn = 自然频率（Natural Frequency）
ζ  = 阻尼比（Damping Ratio）
```

**阻尼比决定了系统行为：**

| 阻尼比 ζ | 系统行为 |
|----------|----------|
| ζ = 0 | 无阻尼，等幅振荡 |
| 0 < ζ < 1 | 欠阻尼，衰减振荡 |
| ζ = 1 | 临界阻尼，最快无超调 |
| ζ > 1 | 过阻尼，慢速响应 |

**关键性能指标：**

```
上升时间 tr：输出从10%上升到90%的时间
峰值时间 tp：达到第一个峰值的时间
超调量 Mp：超出稳态值的最大百分比
调节时间 ts：进入±2%误差带的时间
```

---

### 2.3 频域分析 ⭐⭐⭐⭐⭐

**用频率的视角看系统！**

**波特图（Bode Plot）：**

![波特图](https://upload.wikimedia.org/wikipedia/commons/thumb/5/5d/BodeLowPass.svg/800px-BodeLowPass.svg.png)

- **幅频特性**：|G(jω)| 随频率的变化
- **相频特性**：∠G(jω) 随频率的变化

**稳定裕度：**

| 裕度 | 含义 | 健康值 |
|------|------|--------|
| 相位裕度 PM | 离不稳定还有多少相位余量 | > 45° |
| 幅值裕度 GM | 离不稳定还有多少增益余量 | > 6dB |

### 2.4 PID 控制——工业界的"万金油" ⭐⭐⭐⭐⭐

**PID 是什么？**

PID 控制器是工业界应用最广泛的控制算法，**95%以上的工业控制回路都在用 PID**。

```
u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
       ↑         ↑            ↑
       比例项    积分项       微分项
```

**三个参数的作用：**

| 参数 | 作用 | 太大会怎样 |
|------|------|-----------|
| Kp（比例） | 加快响应，减小误差 | 超调增大，甚至振荡 |
| Ki（积分） | 消除稳态误差 | 响应变慢，超调增大 |
| Kd（微分） | 抑制超调，改善动态 | 对噪声敏感 |

**PID 调参口诀：**

```
比例好比人的反应速度，越大越敏感
积分好比人的耐心，越大越执着
微分好比人的预判，越大越谨慎
```

## 第三阶段：现代控制理论——从经典到前沿

![现代控制](https://images.unsplash.com/photo-1620712943543-bcc4688e7485?w=800)

经典控制理论用传递函数描述系统，而**现代控制理论用状态空间**。

### 3.1 状态空间分析 ⭐⭐⭐⭐⭐

**为什么需要状态空间？**

- 可以描述多输入多输出（MIMO）系统
- 可以用计算机直接求解
- 是最优控制、鲁棒控制的基础

**状态空间表示：**

```
ẋ = Ax + Bu   （状态方程）
y = Cx + Du   （输出方程）

x = 状态向量（描述系统内部状态）
u = 输入向量
y = 输出向量
```

**核心概念：能控性和能观性**

| 概念 | 物理含义 | 数学判断 |
|------|----------|----------|
| 能控性 | 能否通过输入将系统从任意状态转移到另一状态 | rank([B AB A²B ...]) = n |
| 能观性 | 能否通过输出确定系统的初始状态 | rank([C CA CA² ...]ᵀ) = n |

---

### 3.2 极点配置 ⭐⭐⭐⭐

**核心思想：通过状态反馈，把系统极点放到任意位置！**

```
状态反馈控制律：u = -Kx

闭环系统：ẋ = (A - BK)x

通过选择 K，可以任意配置闭环极点！
```

**这意味着什么？**

你可以让系统：
- 响应更快
- 超调更小
- 更加稳定

---

### 3.3 最优控制（LQR）⭐⭐⭐⭐

**核心问题：如何在满足性能要求的同时，消耗最少的控制能量？**

**线性二次型调节器（LQR）：**

```
目标函数：
J = ∫(xᵀQx + uᵀRu)dt

其中：
Q = 状态权重矩阵（希望状态误差小）
R = 控制权重矩阵（希望控制能量小）
```

**LQR 的优势：**

- 有最优解（数学上可证明）
- 计算简单（解 Riccati 方程）
- 鲁棒性好（有一定稳定裕度）

---

### 3.4 卡尔曼滤波——状态估计的利器 ⭐⭐⭐⭐

**问题：传感器的测量总是有噪声，如何获得准确的状态估计？**

**卡尔曼滤波的两步：**

```
预测步：根据模型预测下一状态
x̂(k|k-1) = Ax̂(k-1) + Bu(k)

更新步：根据测量修正估计
x̂(k) = x̂(k|k-1) + K(z(k) - Hx̂(k|k-1))
                              ↑
                         卡尔曼增益
```

**应用场景：**

- 无人机的位置估计
- 自动驾驶的车辆定位
- 机器人的姿态估计

## 第四阶段：数字控制系统——让控制走进计算机

![数字控制](https://images.unsplash.com/photo-1517694712202-14dd9538aa97?w=800)

现实中的控制器几乎都是**数字控制器**（单片机、DSP、嵌入式系统）。

### 4.1 采样定理 ⭐⭐⭐⭐

**香农采样定理：**

```
采样频率 fs ≥ 2 × 信号最高频率 fmax

否则会发生混叠，无法恢复原信号！
```

### 4.2 离散 PID ⭐⭐⭐⭐⭐

**从连续到离散：**

```
连续 PID：                  离散 PID：
u = Kp·e + Ki·∫edt + Kd·de/dt    u = Kp·e + Ki·∑e·Ts + Kd·(e-e_prev)/Ts
        ↓                              ↓
   积分用求和代替              微分用差分代替
```

## 第五阶段：机器人学基础——从理论到实践

![机器人](https://images.unsplash.com/photo-1531746790731-6c087fecd65a?w=800)

### 5.1 刚体运动学 ⭐⭐⭐⭐⭐

**三维空间中的运动描述：**

- **旋转矩阵**：描述刚体的姿态
- **欧拉角**：用三个角度描述姿态（容易理解但有奇异性）
- **四元数**：用四个数描述姿态（无奇异性，计算高效）

### 5.2 正运动学与逆运动学 ⭐⭐⭐⭐⭐

**正运动学：已知关节角度，求末端位置**

```
输入：θ₁, θ₂, θ₃（关节角度）
输出：x, y, z（末端位置）
```

**逆运动学：已知末端位置，求关节角度**

```
输入：x, y, z（期望的末端位置）
输出：θ₁, θ₂, θ₃（需要的关节角度）
```

**逆运动学的难点：**

- 可能无解（目标超出工作空间）
- 可能多解（手臂有多种姿态到达同一点）
- 需要数值迭代求解

### 5.3 轨迹规划 ⭐⭐⭐⭐

**如何让机器人平滑地从A点移动到B点？**

常用方法：
- 多项式插值
- 样条曲线
- 贝塞尔曲线
