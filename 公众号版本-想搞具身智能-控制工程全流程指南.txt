【第一阶段：数学基础——通往控制世界的钥匙】

控制工程的本质是用数学语言描述物理世界。如果你数学基础扎实，后面的学习会事半功倍。

▶ 1.1 线性代数（重要性：⭐⭐⭐⭐⭐）

为什么重要？

现代控制理论几乎全部建立在矩阵运算之上。状态空间表示、能控性能观性分析、最优控制——全都离不开线性代数。

核心内容：
• 矩阵运算 → 状态方程 ẋ = Ax + Bu
• 特征值与特征向量 → 判断系统稳定性
• 矩阵分解 → 卡尔曼滤波、最优控制

学习建议：
• 强烈推荐：3Blue1Brown《线性代数的本质》（B站有中字）
• 教材：Gilbert Strang《线性代数导论》
• 实践：用 Python numpy 库动手算

━━━━━━━━━━━━━━━━━━━━━━

▶ 1.2 微积分与微分方程（重要性：⭐⭐⭐⭐⭐）

为什么重要？

物理世界的运动规律本质上都是微分方程。牛顿第二定律 F=ma 就是一个二阶微分方程。

核心内容：
• 导数与偏导数 → 系统灵敏度分析
• 微分方程 → 系统动力学建模
• 拉普拉斯变换 → 传递函数分析

【拉普拉斯变换——控制工程师的"魔法棒"】

拉普拉斯变换可以把复杂的微分方程变成简单的代数方程：

时域（难算）→ 频域（好算）
微分方程 → 代数方程

常用变换对：
• 1 → 1/s
• t → 1/s²
• e^(-at) → 1/(s+a)
• sin(ωt) → ω/(s²+ω²)

【第二阶段：自动控制原理——控制工程的核心】

自动控制原理是控制工程的入门必修课，也是面试的重点考察内容。

▶ 2.1 系统建模（重要性：⭐⭐⭐⭐⭐）

什么是系统建模？
用数学方程描述物理系统的行为。

经典案例：弹簧-质量-阻尼系统

运动方程：mẍ + cẋ + kx = F(t)
其中：m=质量，c=阻尼系数，k=弹簧刚度

传递函数形式：
G(s) = X(s)/F(s) = 1/(ms² + cs + k)

━━━━━━━━━━━━━━━━━━━━━━

▶ 2.2 时域分析（重要性：⭐⭐⭐⭐⭐）

核心问题：系统输入一个信号后，输出会怎样变化？

【二阶系统（最重要！）】

G(s) = ωn²/(s² + 2ζωns + ωn²)

ωn = 自然频率
ζ = 阻尼比

阻尼比决定了系统行为：
• ζ = 0：无阻尼，等幅振荡
• 0 < ζ < 1：欠阻尼，衰减振荡
• ζ = 1：临界阻尼，最快无超调
• ζ > 1：过阻尼，慢速响应

关键性能指标：
• 上升时间 tr：输出从10%上升到90%的时间
• 峰值时间 tp：达到第一个峰值的时间
• 超调量 Mp：超出稳态值的最大百分比
• 调节时间 ts：进入±2%误差带的时间

━━━━━━━━━━━━━━━━━━━━━━

▶ 2.3 PID 控制——工业界的"万金油"（重要性：⭐⭐⭐⭐⭐）

PID 是什么？

PID 控制器是工业界应用最广泛的控制算法，95%以上的工业控制回路都在用 PID。

u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
↑ 比例项 ↑ 积分项 ↑ 微分项

三个参数的作用：
• Kp（比例）：加快响应，减小误差
• Ki（积分）：消除稳态误差
• Kd（微分）：抑制超调，改善动态

【PID 调参口诀】

比例好比人的反应速度，越大越敏感
积分好比人的耐心，越大越执着
微分好比人的预判，越大越谨慎

【第三阶段：现代控制理论——从经典到前沿】

经典控制理论用传递函数描述系统，而现代控制理论用状态空间。

▶ 3.1 状态空间分析（重要性：⭐⭐⭐⭐⭐）

为什么需要状态空间？
• 可以描述多输入多输出（MIMO）系统
• 可以用计算机直接求解
• 是最优控制、鲁棒控制的基础

状态空间表示：
ẋ = Ax + Bu（状态方程）
y = Cx + Du（输出方程）

核心概念：
• 能控性：能否通过输入将系统从任意状态转移到另一状态
• 能观性：能否通过输出确定系统的初始状态

━━━━━━━━━━━━━━━━━━━━━━

▶ 3.2 最优控制（LQR）（重要性：⭐⭐⭐⭐）

核心问题：如何在满足性能要求的同时，消耗最少的控制能量？

线性二次型调节器（LQR）：
目标函数：J = ∫(xᵀQx + uᵀRu)dt

LQR 的优势：
• 有最优解（数学上可证明）
• 计算简单（解 Riccati 方程）
• 鲁棒性好（有一定稳定裕度）

━━━━━━━━━━━━━━━━━━━━━━

▶ 3.3 卡尔曼滤波——状态估计的利器（重要性：⭐⭐⭐⭐）

问题：传感器的测量总是有噪声，如何获得准确的状态估计？

卡尔曼滤波的两步：
1. 预测步：根据模型预测下一状态
2. 更新步：根据测量修正估计

应用场景：
• 无人机的位置估计
• 自动驾驶的车辆定位
• 机器人的姿态估计

━━━━━━━━━━━━━━━━━━━━━━

【第四阶段：数字控制系统——让控制走进计算机】

现实中的控制器几乎都是数字控制器（单片机、DSP、嵌入式系统）。

▶ 4.1 采样定理（重要性：⭐⭐⭐⭐）

香农采样定理：
采样频率 fs ≥ 2 × 信号最高频率 fmax
否则会发生混叠，无法恢复原信号！

▶ 4.2 离散 PID（重要性：⭐⭐⭐⭐⭐）

从连续到离散：
• 积分 → 用求和代替
• 微分 → 用差分代替

━━━━━━━━━━━━━━━━━━━━━━

【第五阶段：机器人学基础——从理论到实践】

▶ 5.1 刚体运动学（重要性：⭐⭐⭐⭐⭐）

三维空间中的运动描述：
• 旋转矩阵：描述刚体的姿态
• 欧拉角：用三个角度描述姿态
• 四元数：用四个数描述姿态（无奇异性）

▶ 5.2 正运动学与逆运动学（重要性：⭐⭐⭐⭐⭐）

正运动学：已知关节角度，求末端位置
逆运动学：已知末端位置，求关节角度

逆运动学的难点：
• 可能无解（目标超出工作空间）
• 可能多解（手臂有多种姿态）
• 需要数值迭代求解
