## 一、刚体运动学

### 1.1 旋转矩阵

**绕 X/Y/Z 轴旋转：**

```
Rx(θ) = [1    0      0  ]
        [0  cosθ  -sinθ]
        [0  sinθ   cosθ]

Ry(θ) = [cosθ   0  sinθ]
        [  0    1    0 ]
        [-sinθ  0  cosθ]

Rz(θ) = [cosθ  -sinθ  0]
        [sinθ   cosθ  0]
        [  0      0   1]
```

### 1.2 欧拉角与四元数

| 欧拉角 (Roll, Pitch, Yaw) | 四元数 q = [w, x, y, z] |
|---------------------------|-------------------------|
| 直观易懂 | 无奇异性 |
| 有万向锁问题 | 计算高效 |
| 插值困难 | 插值平滑 |

## 二、正运动学与逆运动学

### 2.1 正运动学

**定义：** 已知关节角度 → 计算末端位置

**2DOF 机械臂：**

```
x = L₁·cos(θ₁) + L₂·cos(θ₁+θ₂)
y = L₁·sin(θ₁) + L₂·sin(θ₁+θ₂)
```

### 2.2 逆运动学

**定义：** 已知末端位置 → 计算关节角度

**难点：**
- 可能多解
- 可能无解（超出工作空间）
- 可能有奇异点

## 四、机器人动力学

### 4.1 拉格朗日方程

```
L = T - V （动能 - 势能）

d/dt(∂L/∂q̇) - ∂L/∂q = τ
```

### 4.2 牛顿-欧拉方法

